# Pacman2

## Description

Link to the CS188 course: [Pacman 2](https://inst.eecs.berkeley.edu/~cs188/fa25/projects/proj1/#q1-3-pts-finding-a-fixed-food-dot-using-depth-first-search)

## Dependencies

- Python 3.x
- Tkinter

## Usage

To run the Pacman game from the root directory, execute the following command in your terminal:

```bash
cd multiagent && python pacman.py
```
